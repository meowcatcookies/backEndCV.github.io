專業經歷與技術應用

  • 系統與職責： 負責開發與維護後勤管理系統的後端服務，此系統旨在提升勞力密集型產業的協作效率，主要功能包含權限分級、任務自動指派、資料分析與報表生成。
  • 技術優化：
    • 將服務改為依賴注入 (Dependency Injection) 進行管理，並將同步方法重構為非同步方法，顯著提升應用程式響應速度。
    • 將低效的 Entity Framework 查詢替換為高效的 Dapper，並搭配非叢集索引優化資料庫查詢。
    • 運用 .NET IMemoryCache 實作快取機制，提升常用資料的讀取效能；同時利用快取處理黑名單邏輯，確保系統穩定性與安全性。

  • API 開發與文件： 參與新專案後端 API 開發，採用前後端分離架構。使用 Swagger 建立 OpenAPI 規格文件，並透過 summary 註解自動生成詳細 API 文件，大幅提升與前端團隊的協作效率。
  • 欄位權限管理： 實現欄位級權限控制，使系統可根據使用者權限動態顯示不同欄位，賦予業主高度客製化能力。
  • 資料庫操作抽象化： 嘗試透過反射 (Reflection) 技術，利用 Class 物件自動組建 SQL 語法進行 CRUD 操作。

  • 效能瓶頸分析： 透過 Azure SQL 與 SSMS 的「最耗效能的查詢」報告，定位並優化高負載查詢，確保邏輯精簡。
  • 語法與架構重構： 運用 CTE (Common Table Expressions) 等 T-SQL 語法重構複雜查詢，並設計「每日彙總資料表」，將多個關聯表的資料預先彙總，大幅提升查詢效能。
  • 顯著成果： 成功將 Azure SQL 的服務等級從 V6 降低至 V2，有效節省成本並解決 CPU 負載超過 90% 的尖峰問題。

  • 部署與配置安全： 實施部署連線字串加密，並將敏感資訊設定為需讀取環境變數才能存取。
  • 前端安全性強化： 升級所有過時的前端依賴套件，並在 HTTP 標頭中加入 Content Security Policy (CSP) 規範，防止網站被 iframe 嵌入。同時強化 JavaScript 程式碼以防禦 XSS 攻擊。
